# ProtfolioPro - Personal Portfolio Website

A modern, responsive personal portfolio website showcasing skills, projects, and professional experience.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices (desktop, tablet, mobile)
- **Modern UI/UX**: Clean and professional design with smooth animations and transitions
- **Interactive Navigation**: Smooth scrolling navigation with active state indicators
- **Sections Include**:
  - Hero section with animated typing effect
  - About Me with professional statistics
  - Skills showcase with technology cards
  - Featured projects with links
  - Contact form with social media links
  - Professional footer

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- JavaScript (Vanilla JS)
- Font Awesome Icons
- Modern CSS animations and transitions

## Getting Started

1. Clone this repository
2. Open `index.html` in your web browser
3. Customize the content with your personal information

## Customization

### Update Personal Information

Edit the following in `index.html`:

- **Name**: Replace "Your Name" in the hero section and footer
- **Title**: Update the hero subtitle with your professional title
- **About**: Modify the about section text with your background
- **Skills**: Add or remove skill cards as needed
- **Projects**: Update project cards with your actual projects
- **Contact**: Add your email, phone, location, and social media links

### Customize Colors

Edit the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    /* Modify other colors as needed */
}
```
